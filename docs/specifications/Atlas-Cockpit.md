# Atlas Cockpit — Spécification

**Cycle :** Genesis  
**Version :** 0.4.0

## Objectif

Créer le premier poste de pilotage visuel d'Atlas après la sélection de l'avatar.

## Principes

- le Digital Twin reste au centre ;
- les modules gravitent autour de lui ;
- une région corporelle est directement interactive ;
- les données prioritaires restent visibles sans ouvrir de menu ;
- les recommandations distinguent observation, interprétation et proposition ;
- le design doit être futuriste, mais lisible et apaisant.

## Modules initiaux

- Santé ;
- Entraînement ;
- Renforcement ;
- Mobilité ;
- Sommeil ;
- Nutrition ;
- Chronologie ;
- Progression ;
- Prévention ;
- Atlas AI.

## Statut de la version

Cette version est un prototype autonome. Elle n'est pas encore reliée au `index.html` public ni aux moteurs Genesis 0.3.0.
