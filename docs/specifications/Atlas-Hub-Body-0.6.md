# Atlas Hub Body — Genesis 0.6.0

Cette version reconstruit complètement le prototype pour se rapprocher de l'interface de référence validée.

## Changements majeurs

- composition horizontale compacte ;
- avatar central dans un hub circulaire ;
- modules disposés en orbite ;
- colonne d'indicateurs à gauche ;
- panneau Atlas Observe à droite ;
- douleurs récentes ;
- champ Demander à Atlas ;
- chronologie horizontale ;
- six cartes statistiques inférieures ;
- navigation latérale complète ;
- réduction importante du défilement vertical sur écran large.

## Remplacement

Les trois fichiers du dossier `prototype/hub-body` remplacent ceux de Genesis 0.5.0.
