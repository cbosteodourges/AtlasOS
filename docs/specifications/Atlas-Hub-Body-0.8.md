# Atlas Hub Body — Genesis 0.8.0

## Objectif

Obtenir une reproduction visuelle quasiment exacte de l'image validée.

## Méthode

L'image validée devient la couche visuelle de référence. Des zones HTML transparentes et interactives sont positionnées précisément au-dessus :

- modules du hub ;
- points corporels ;
- douleurs récentes ;
- Demander à Atlas ;
- chronologie.

## Avantages

- proportions exactement conservées ;
- avatar identique à la référence ;
- absence de déformation CSS ;
- adaptation automatique à la taille de la fenêtre ;
- interactions conservées.

## Étape suivante

Les panneaux seront progressivement remplacés par de vrais composants dynamiques sans modifier l'apparence visuelle.
